About RepositoryNook UI
This is a JSON data tool, for input, edit and reading JSON data from a NoSQL database.

Design courtesy of Google's material design components and specification.

You configure the UI to connect to an on-prem, or cloud-hosted micro service called: RepositoryNook

  • App (RepositoryNook UI) is written in Typescript & Angular 8. source code (github)
  • Service (RepositoryNook) runs containerized, written in .NET Core 3.0 in C#. source code (github)
  • RESTful API doc created with OpenAPI(Swagger).
  • NoSQL Repository data stored in MongoDB hosted on Atlas (cloud)
  • Auth (OAuth2) by hosted provider www.auth0.com
  • Instrumentation plumbed into Azure Monitor along with call exceptions and API usage data.
