About
Responsive UI App example; leverages material design's components and stylistics.
Connects to a cloud-hosted, and containerized micro service called: RepositoryNook

  • App is written in Typescript & Angular 8. micro service code (github)
  • Service is containerized .NET Core 3.0 in C#. UI code (github)
  • RESTful API doc via OpenAPI/Swagger. Access page here.
  • NoSQL Repository data stored in MongoDB hosted on Atlas (cloud)
  • Auth (OAuth2) by hosted provider www.auth0.com
